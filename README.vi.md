## 💡 Vấn đề: Tại sao các mô hình AI thông minh lại viết mã xấu
Các mô hình AI tiên tiến (chẳng hạn như GPT-4o, Claude 3.5 Sonnet và Gemini Pro) có khả năng suy luận rất lớn. Nhưng nếu không có các nguyên tắc kiến ​​trúc nghiêm ngặt, hộp cát thư mục rõ ràng và các quy tắc định dạng thống nhất, chúng sẽ viết mã phi cấu trúc, bỏ qua các quy ước hiện có của bạn và lặp lại các lỗi tương tự qua các phiên.

---

## ⚡ Giải pháp: Quản trị AI tiêu chuẩn vàng

### 1. 📜 Quy tắc biên tập chéo phổ biến
Đưa các lệnh AI hiệu suất cao (tương đương `AGENTS.md` và `.cursorrules`) tương thích với tất cả các trình soạn thảo AI chính—**Cursor, Windsurf, Claude Code, GitHub Copilot và Kiro**. Một bộ quy tắc để cai trị tất cả.

### 🧠 2. Trình quản lý bộ nhớ độc lập
Bao gồm các tập lệnh stdlib Python gốc giúp tự động tích lũy kiến thức dự án cục bộ. Nếu AI giải quyết được lỗi hoặc đưa ra quyết định về mặt kiến ​​trúc một lần, nó sẽ ghi lỗi đó vào `.agent/memory/` và không bao giờ quên.

### ⚡ Trình cài đặt không phụ thuộc
Không `npm install`, không `pip install`. Cài đặt trong 1 giây thông qua một lớp lót Curl/Powershell gốc đơn giản, chỉ tải xuống các tập lệnh Python trong thư viện tiêu chuẩn và quy trình làm việc đánh dấu rõ ràng.

## 📂 Bản đồ thư mục trực quan

```
your-project/
  .agent/
    scripts/           # Zero-dependency Python tools (Search, Save, Validate)
    workflows/         # Step-by-step markdown workflows (Maintenance, Save)
  AGENTS.md            # The master AI system instructions (entry point)
  install.sh           # Native Linux/macOS installer
  install.ps1          # Native Windows PowerShell installer
```

---

## 🚀 Cài đặt (1 giây)

### Windows (PowerShell):

```powershell
iwr -useb https://raw.githubusercontent.com/andiupn/andy-universal-agent-rules/main/install.ps1 | iex
```

###Linux/macOS (Bash):

```bash
curl -fsSL https://raw.githubusercontent.com/andiupn/andy-universal-agent-rules/main/install.sh | bash
```
