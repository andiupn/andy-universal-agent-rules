## 💡 Проблема: почему умные модели ИИ пишут плохой код
Продвинутые модели искусственного интеллекта (такие как GPT-4o, Claude 3.5 Sonnet и Gemini Pro) обладают огромной способностью к рассуждению. Но без строгих архитектурных рекомендаций, четкой изолированной программной среды для папок и унифицированных правил форматирования они пишут неструктурированный код, игнорируют существующие соглашения и повторяют одни и те же ошибки в разных сеансах.

---

## ⚡ Решение: золотой стандарт управления ИИ

### 1. 📜 Универсальные правила кросс-редактора
Внедряет высокопроизводительные инструкции искусственного интеллекта (эквивалент `AGENTS.md` и `.cursorrules`), совместимые со всеми основными редакторами искусственного интеллекта — **Cursor, Windsurf, Claude Code, GitHub Copilot и Kiro**. Один набор правил, чтобы управлять всеми.

### 🧠 2. Автономные кураторы памяти
Включает встроенные сценарии Python stdlib, которые автоматизируют накопление знаний о локальном проекте. Если ИИ однажды устраняет ошибку или принимает архитектурное решение, он записывает его в `.agent/memory/` и никогда не забывает.

### ⚡ Установщик с нулевой зависимостью
Нет `npm install`, нет `pip install`. Устанавливается за 1 секунду с помощью простого встроенного однострочника Curl/Powershell, загружая только скрипты Python из стандартной библиотеки и рабочие процессы с чистой уценкой.

## 📂 Визуальная карта каталогов

```
your-project/
  .agent/
    scripts/           # Zero-dependency Python tools (Search, Save, Validate)
    workflows/         # Step-by-step markdown workflows (Maintenance, Save)
  AGENTS.md            # The master AI system instructions (entry point)
  install.sh           # Native Linux/macOS installer
  install.ps1          # Native Windows PowerShell installer
```

## 🚀 Установка (1 секунда)

### Windows (PowerShell):

```powershell
iwr -useb https://raw.githubusercontent.com/andiupn/andy-universal-agent-rules/main/install.ps1 | iex
```

### Linux/macOS (Bash):

```bash
curl -fsSL https://raw.githubusercontent.com/andiupn/andy-universal-agent-rules/main/install.sh | bash
```
